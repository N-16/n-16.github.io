03D

Solo Project

03D is a first-person perspective action platformer project, which is based on the idea of recreating the mechanics of one of my favorite games Katana Zero in a 3D environment.

 
Roles Description
Program Programmed for the entire project using Unity Game Engine. Notable Sections: Player Mechanics, Enemy AI (State Machine), Combat.
Level Design Designed and created levels using Unity's ProBuilder.
Other Cel Shader (using Shader Graph), Sound Design.

 

Assets used:

- Enemy Model and Animations: Mixamo

- Sword Model

- Textures for Floor and Walls

- Footstep sounds
