03D

Solo Project

03D is a first person perspective action platformer project, which is based on the idea of recreating mechanics of one of my most favorite games, Katana Zero in a 3D environment.

 
Roles Description
Program Programmed for the entire project using Unity Game Engine. Notable sections : Player Movement, Enemy AI (State Machine), Combat.
Level Design Designed and created level, using unity's probuilder
Other Cel shader for player and enemy (using shadergraph).
