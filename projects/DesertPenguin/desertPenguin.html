Desert Penguin

Solo Project

Desert Penguin is a 2D endless runner game, in which the player has to dodge bombs and arrow bombs spawned by an aeroplane and survive. The aeroplane may also spawn 'glitches' that may add to the chaos of the game. This game was my submission for the 'HackerEarth's Game Jam 1.0'.

 
Roles Description
Program Programmed for entire game using Unity engine.
Other All the sounds were samples from freesound.org and were slightly edited on fl studio. Design. UI.

Art Assets used: Aeroplane sprite and Stars on head animation.
