Desert Penguin

Solo Project

Desert Penguin is a 2D endless runner game, in which the player (the penguin) has to dodge bombs and arrow bombs spawned by an aeroplane while moving forward, to avoid dark zone and survive. The aeroplane may also spawn 'glitches' (Headblock: Spawns a block on random jumps, Fireboots: Reduces health when player walks over covered ground, Dig: Random Jumps towards opposite direction) that adds to the chaos of the game. This game was my submission for the 'HackerEarth's Game Jam 1.0'.

 
Roles Description
Program Programmed for entire game using Unity engine. Notable Sections: Player Mechanics, Glitch Mechanics, Bomb Spawner/Mechanics, Level Generator
