Base Dolphin

Solo Project

Base Dolphin is a 3D audio only shooter game where the player has to detect the location of targets using the radar which plays a notification sound panned to the direction of targets, to aim and shoot them down. This game was a submission for the 'Game for Blind Gamers' Game Jam.

 
Roles Description
Program Programmed for entire game using Unity engine. Notable Sections: Target (Submarines) spawner and path, radar, audio only sensitivity tester/adjuster.
Sounds All the sound effects were made using various samples and virtual instruments on fl studio. Sounds for announcements were made using sampled text-to-speech.
Other Design.
