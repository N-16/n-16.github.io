Space Rings

Solo Project

Space Rings is a 3D endless runner game in which the player has to drive the spaceship through the rings, while avoiding obstacles on the way. It was made for the final assignment of CS50's Game Development course.

 
Roles Description
Program Programmed for entire game using Unity engine. Notable Sections: Space Ship Movement/Controller, Level Spawner, Obstacle Mechanics.
Sounds and Music All the sounds were made using samples or virtual instruments on fl studio. The same tool was used to produce music for the game.
Models All the models were made using unity's built in probuilder.
Other Design, UI.
