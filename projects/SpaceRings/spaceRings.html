Space Rings

Solo Project

Space Rings is a 3D endless runner game in which the player controls a spaceship, and in order to keep it charged/moving, the player has to drive it through the rings that are placed at certain intervals. There are also two types of obstacles placed in between rings which makes it difficult for the player to follow the rings.

This project was my submission for the final assignment of CS50's Game Development course.

 
Roles Description
Program Programmed for the entire game using Unity engine. Notable Sections: Spaceship Controller, Level Spawner, Obstacle Mechanics.
Sounds and Music All the sounds were made using samples or virtual instruments on FL Studio. The same tool was used to produce music for the game.
Models All the models were made using Unity's built-in ProBuilder.
Other Design, UI.
